# 🎓 Scholarship Hub — Centralized Scholarship Discovery Portal

> A frontend web portal developed during an internship at **NIT Warangal (NITW)**, designed to simplify scholarship discovery for students across India by consolidating opportunities from multiple sources into a single, accessible platform.

---

## 📌 Project Overview

Students in India often struggle to track scholarships scattered across government portals, university websites, and private organizations. This project addresses that problem by building a **Scholarship Hub** — a centralized informational portal where students can explore, discover, and understand scholarship opportunities from one place.

Rather than visiting dozens of different websites, students can visit this portal and find categorized, organized, and clearly presented scholarship information — all in one place.

## ✨ Features

### 🏠 Home Page
- Navigation bar with dropdown menu for scholarship categories
- Image slider showcasing key scholarship highlights
- Scholarship category cards (International, National, State, Private)
- Search bar for quick scholarship lookup
- About section describing the platform's purpose

### 📂 Scholarship Category Pages
Dedicated pages for each scholarship type:
- 🌍 **International** — Study abroad and foreign-funded scholarships
- 🇮🇳 **National** — Government of India funded scholarships (UGC, AICTE, INSPIRE, etc.)
- 🏛️ **State** — State-specific post-matric and merit-based scholarships
- 🏢 **Private** — Corporate and NGO-funded scholarship opportunities

### 📢 Announcement Section
Three-panel announcement board:
- **Upcoming** — Scholarships opening soon
- **Current** — Active and ongoing scholarships
- **Previous** — Archived past scholarships for reference

### 📅 Timeline / Workflow Section
Step-by-step scholarship application workflow outlining responsibilities of:
- Students
- Colleges / Institutions
- District Officers

### 🌐 Multilingual Support
- **Google Translate** integration supporting multiple Indian regional languages
- Improves accessibility for non-English speaking students across India

### 🆘 Helpdesk
- FAQ page addressing common scholarship queries
- Contact page with support information

### 🦶 Footer
- Contact information
- About platform summary
- Social media links

### 🎨 UI / UX
- Responsive card-based layout
- Scrolling announcements with hover-to-pause interaction
- Dropdown navigation menus
- Light / Dark mode toggle
- Mobile-friendly responsive design

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **HTML5** | Semantic page structure and content |
| **CSS3** | Styling, Flexbox, Grid, animations, responsive design |
| **JavaScript (ES6)** | DOM manipulation, theme toggle, slider controls |
| **Google Translate API** | Multilingual support for Indian regional languages |
| **Font Awesome** | Icons used across the UI |
| **Responsive Web Design** | Mobile-first approach with media queries |

> **Note:** This is a purely frontend, static web portal. There is no backend, database, authentication, or server-side functionality.
